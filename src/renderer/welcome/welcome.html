desktop assistant

Understand anything on your screen while you work.

ScreenSense AI gives you a professional floating assistant for live captions, screen explanations, transcripts, summaries, and presentation-ready notes without forcing you to leave the app you are using.

Live Captions

Transcribe spoken audio in real time and keep important lines visible while meetings, videos, or lessons are playing.

Scene Explain

Transcript Notes

Turn speech into cleaner notes so long sessions become easier to review, export, and share.

Summarizer

Collect audio and visual context, then export a professional Markdown summary with key points and timelines.

Face Analysis

Analyze visible expressions and presentation tone when your workflow needs audience or speaker context.

Exports

Save session notes, summaries, and generated outputs directly to Downloads for follow-up work.

Session Chat

Ask questions about the active session while ScreenSense uses recent captions, screen observations, and saved context to answer without leaving your workflow.

Step 1

Set up API keys for ScreenSense AI.

An API key is a secure code that lets ScreenSense connect to an AI provider from your own account. Live Captions run locally with Whisper and do not require a cloud API key. For cloud AI features such as chat, summaries, translation, cleanup, and fallback providers, start with Groq because it is the simplest option to add in Settings.

Recommended: open the Groq Console, sign in, go to API Keys, create a new key, and copy the key that starts with gsk_.
Open ScreenSense Settings, paste the Groq key into the Groq API Key field, and save your settings.
Optional: open Google AI Studio for Gemini, create a key, and store it in .env.local as GOOGLE_AI_STUDIO_API_KEY=your_key if you use environment-based provider keys.
Optional: use OpenAI API Keys or the Anthropic Console only when you need those provider-specific features.
Keep every key private. Do not post it online, send it in chat, or share screenshots where the key is visible.
