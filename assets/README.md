# Assets

Place your app icons here before building:

- icon.png  — 512×512 PNG (source image, required for Linux)
- icon.ico  — Windows icon (convert from icon.png at https://convertio.co)
- icon.icns — macOS icon (convert from icon.png at https://cloudconvert.com)

To generate all formats automatically from icon.png, run:
  npx electron-icon-maker --input=assets/icon.png --output=assets/
